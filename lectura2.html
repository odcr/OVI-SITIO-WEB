ETIQUETAS HTML5

< !DOCTYPE >:

Esta es la primer línea que debe aparecer en nuestro código para que el navegador se active de forma estándar con sus incorporaciones. Esta será la declaración del tipo de documento que se va a mostrar.

< HTML >:

Define la raíz del documento HTML, este elemento envolverá a todo el código.

< HEAD >:

Define información acerca de un documento. En este paso se definen los títulos que se presentaran en la página.







< BODY >:

Aquí se define el cuerpo del documento. Lo que va a ser visible para el usuario en la página web.>

< META >:

Aquí se define un metadato de un documento HTML, significa como será mostrado el texto en la pantalla de la página web.

< TITLE > :

Se define un título para el documento que se está creando. Este texto es mostrado en la barra superior de la ventana del navegador.

< LINK > :

Aquí se define la relación entre un documento y un recurso externo tal es el caso donde se incorporan estilos, códigos JavaScript, imágenes o archivos desde sitios externos.
