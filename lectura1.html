¿ QUE ES HTML ?



Es una nueva versión de HTML, con nuevos elementos que son de gran interés para las personas que gustan de la programación o solo buscan mejorar sus conocimientos. El HTML5 provee tres características principales: la estructura, el estilo y su funcionalidad. El HTML5 es considerado como un producto de combinación entre HTML y CSS y Javascript.

El HTML5 es usado para crear sitios webs permitiendo manejar con mayor facilidad su plataforma y el trabajo de quien lo realiza. Además una de las ventajas del HTML5 es que es compatible con todos los navegadores actuales, la estructura de etiquetas mejoradas donde permite trabajar por separado las secciones que componen la página web. Como también se puede incluir videos e imágenes para su reproducción y que estas funcionen sin errores cuando se muestren al público.

En si el lenguaje ofrece una serie de ventajas permitiendo tener un código más accesible, limpio y fácil de usar. Algo que es muy evidente y que cabe resaltar en HTML5 a diferencia de otros lenguajes es que no necesita el cierre de etiquetas, se pueden usar mayúsculas como minúsculas en el código, etc.
